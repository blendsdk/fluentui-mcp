## Import Patterns

### Main Package Import

```typescript
// Import from main package (recommended)
import {
  Button,
  Input,
  Dialog,
  // ... other components
} from '@fluentui/react-components';
```

### Individual Package Import

```typescript
// For bundle optimization, import from individual packages
import { Button } from '@fluentui/react-button';
import { Input } from '@fluentui/react-input';
```

## Quick Component Selection Guide

| Need | Use Component |
|------|---------------|
| User action | Button, MenuButton, SplitButton |
| Text input | Input, Textarea, SearchBox |
| Selection (single) | Radio, Select, Combobox |
| Selection (multiple) | Checkbox, TagPicker |
| Boolean toggle | Switch, Checkbox, ToggleButton |
| Numeric input | SpinButton, Slider |
| Display user | Avatar, Persona |
| Status indicator | Badge, Spinner, ProgressBar |
| Show message | MessageBar, Toast, Dialog |
| Additional info | Tooltip, Popover, TeachingPopover |
| Navigation | Menu, Tabs, Breadcrumb, Nav, Link |
| Side navigation drawer | NavDrawer, Nav |
| List data | Table, List, Tree |
| Group content | Card, Accordion |
| Slide-out panel | Drawer, NavDrawer |
